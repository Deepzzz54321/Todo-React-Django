About

TodoEPC is a simple app to keep track of your todos. You get to add, edit, mark as complete/incomplete, delete your todos. But it won't stop there. It is and will continue to evlove to make your life easier everyday.

The app, in software terms, could be considered as a basic CRUD(Create-Read-Update-Delete) app. Falling under the client-server model, the client(React) makes API calls to the server(Django REST framework), fetches the todos and presents them while the server(Django REST framework) exposes API endpoints, handles the authentication, communicates with the database.

Tech Stack
  • Django REST framework
  • ReactJS
  • Bootstrap
  • Axios
  • React Toastify
Concepts Implemeted
  • Django+React Integration
  • API Implementation
  • Session Authentication
  • Email Verification
